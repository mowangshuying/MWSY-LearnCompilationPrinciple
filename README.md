### 授课的顺序

![image-20220507222835705](./img/image-20220507222835705.png)

* 一学期的课程可包含第一部分的所有章节，同时让学生实现编译器，多半按照项目组的方式进行。
* 高级课程或者研究生课程可包含第二部分内容
* 按两个半学习安排教学的话，前半个学期包含1-8章，后半个学期包括9-12章和第二部分的某些章
